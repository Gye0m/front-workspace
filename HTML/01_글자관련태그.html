그 밖의 글자 관련 문장태그들

strong태그는 글자를 굵게 표시합니다.
b : 글자를 굵게(bold)표시하는 태그

em : 글자를 기울여서 표시하는 태그

i : 글자를 기울여서 표시하는 태그 mark : 형광펜, 글자를 강조하는 태그

u : 글자에 밑줄이 그어지는 태그

ins : 글자에 밑줄이 그어지는 태그

글자에 취소선을 그어주는 태그

del : 글자에 취소선이 그어지는 태그

이건 큰 글자임small : 글자를 작게 표시하는 태그

이건 보통 글자임big : 글자를 크게 표시하는 태그

IOT 란 각종 사물에 센서와 통신기능을 탑재해 인터넷에 연결하는 기술을 의미한다.

sub : 기본글자에 아래첨자를나타내는 태그

sup : 기본글자에 위첨자를나타내는 태그

code : 코드블럭을 나타내는 태그 System.out.println("Hello, World!");








글자관련 태그 응용1

태그들은 해당 태그내에서 중첩으로 사용가능하다. 굵은글자를 사용할 수도 있고, 기울이거나,취소선을 넣는 등 다양하게 사용할 수 있다.
형광펜을 넣어 글자를 강조할 수 있다. 정말 중요한 내용의 경우 태그들을 여러개 중첩하기도한다.
